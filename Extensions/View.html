extension View

Generic extension to apply conditional transforms.

Source: https://www.avanderlee.com/swiftui/conditional-view-modifier/

  • if(_:transform:)

    Applies the given transform if the given condition evaluates to true.

    Declaration

    Swift

    @ViewBuilder
@MainActor
public func `if`<Content>(_ condition: Bool, @ViewBuilder transform: (Self) -> Content) -> some View where Content : View

    Parameters

    condition

    The condition to evaluate.
    transform

    The transform to apply to the source View.

    Return Value

    Either the original View or the modified View if the condition is true.
